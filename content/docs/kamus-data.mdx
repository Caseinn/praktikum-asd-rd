Kamus Data adalah komponen fundamental dalam pemrograman yang digunakan untuk mendeklarasikan dan mendefinisikan nama-nama yang akan digunakan dalam program.  
**Kamus Data** berisi deklarasi type, variabel, dan konstanta yang dibuat oleh programmer.  
**Tipe Data** menentukan jenis nilai yang dapat disimpan dan operasi yang dapat dilakukan pada data tersebut.

Pemahaman yang baik tentang kamus data dan tipe data adalah langkah pertama dalam menulis program yang efisien dan mudah dipahami.

---

## Tujuan Pembelajaran

Setelah mempelajari materi ini, mahasiswa diharapkan mampu:

- **Memahami konsep kamus data** sebagai tempat deklarasi nama-nama dalam program dan membedakannya dari instruksi program.
- **Mengenal tipe data primitif** yang tersedia dalam bahasa pemrograman, termasuk karakteristik dan penggunaan masing-masing tipe.
- **Memahami tipe data turunan** dan cara membentuk struktur data yang lebih kompleks dari tipe data dasar.
- **Menerapkan deklarasi yang tepat** sesuai dengan kebutuhan program dan mengikuti konvensi penamaan yang baik.

---

## Mengapa Materi Ini Penting?

1. **Dasar Pemrograman**  
   Setiap program dimulai dengan deklarasi data. Tanpa pemahaman yang baik tentang kamus data dan tipe data, programmer akan kesulitan merancang struktur program yang efisien dan mudah dipelihara.

2. **Efisiensi Memori**  
   Pemilihan tipe data yang tepat menentukan penggunaan memori program. Menggunakan `int` untuk data yang sebenarnya cukup dengan `char` akan membuang-buang memori, terutama dalam aplikasi berskala besar.

3. **Pencegahan Error**  
   Deklarasi yang jelas dan konsisten membantu mencegah kesalahan logika dan memudahkan proses debugging. Kompiler dapat mendeteksi ketidakcocokan tipe data sejak awal.

## Konsep Kamus Data

### Definisi
Kamus data digunakan untuk mendeklarasikan nama-nama yang digunakan dalam program, meliputi:
- **Type**: Definisi tipe data baru
- **Variabel**: Tempat penyimpanan data yang nilainya dapat berubah
- **Konstanta**: Nilai tetap yang tidak dapat diubah selama program berjalan

### Format Deklarasi
```cpp
[tipe_data] [nama];
```

**Contoh:**
```cpp
int i;                // Variabel integer
float jumlahUang;     // Variabel bilangan desimal
```

**Catatan Penting**: Deklarasi adalah definisi nama, bukan instruksi yang dieksekusi.

---

## Tipe Data

### Tipe Data Primitif
Tipe data dasar yang tersedia dalam C++:

- **`bool`** - Boolean (true/false)
- **`int`** - Integer (bilangan bulat)
- **`float`** - Real (bilangan desimal)
- **`char`** - Character (karakter tunggal)
- **`string`** - String (rangkaian karakter)

### Tipe Data Turunan
Dibentuk dari kombinasi tipe data primitif untuk merepresentasikan entitas yang lebih kompleks.

**Contoh Struktur DataMahasiswa:**
```cpp
struct DataMahasiswa {
    string NIM;
    string Nama;
    int Umur;
    string Kota;
};
```

**Contoh Array:**
```cpp
int umur[10];  // Array 10 bilangan integer untuk data umur
```
