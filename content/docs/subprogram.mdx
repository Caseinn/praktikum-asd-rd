Sub Program adalah blok program yang berisi sekumpulan instruksi untuk menyelesaikan tugas khusus yang dapat dipanggil berulang kali.  
**Sub Program** membantu memecah program kompleks menjadi bagian-bagian yang lebih kecil, mudah dikelola, dan dapat digunakan kembali.  
**Modularitas** adalah prinsip pemrograman yang membagi program besar menjadi modul-modul kecil yang memiliki fungsi spesifik.

Tanpa sub program, kode yang sama harus ditulis berulang-ulang, membuat program tidak efisien dan sulit dipelihara.

---

## Tujuan Pembelajaran

Setelah mempelajari materi ini, mahasiswa diharapkan mampu:

- **Memahami konsep sub program** sebagai solusi untuk mengurangi redundansi kode dan meningkatkan efisiensi program.
- **Membedakan fungsi dan prosedur** serta memahami kapan menggunakan masing-masing tipe sub program.
- **Mendeklarasikan dan menggunakan parameter** untuk komunikasi data antara program utama dan sub program.
- **Menerapkan passing parameter** dengan teknik pass by value dan pass by reference sesuai kebutuhan.

---

## Mengapa Penting?

1. **Efisiensi Kode**  
   Menghindari duplikasi kode yang sama. Daripada menulis ulang 50 baris kode yang sama, cukup panggil fungsi yang sudah dibuat. Ini menghemat waktu development dan mengurangi ukuran program.

2. **Kemudahan Maintenance**  
   Jika ada bug atau perlu perubahan logika, cukup diperbaiki di satu tempat (dalam fungsi) dan akan berdampak pada semua bagian yang menggunakan fungsi tersebut.

3. **Reusabilitas**  
   Fungsi yang sudah dibuat dapat digunakan di program lain. Misalnya, fungsi untuk menghitung luas lingkaran dapat digunakan di berbagai program geometri.

4. **Keterbacaan Kode**  
   Program menjadi lebih terstruktur dan mudah dipahami karena setiap fungsi memiliki tugas yang spesifik dan nama yang deskriptif.

## Masalah Tanpa Sub Program

Bayangkan program yang menghitung luas berbagai bentuk:

```cpp
// Tanpa sub program - kode berulang dan tidak efisien
int main() {
    // Hitung luas lingkaran 1
    float r1 = 5;
    float luas1 = 3.14159 * r1 * r1;
    
    // Hitung luas lingkaran 2
    float r2 = 7;
    float luas2 = 3.14159 * r2 * r2;
    
    // Hitung luas lingkaran 3
    float r3 = 10;
    float luas3 = 3.14159 * r3 * r3;
    
    // ... berulang untuk setiap perhitungan
}
```

**Masalah:**
- Kode duplikasi yang tidak perlu
- Sulit dipelihara jika formula berubah
- Program menjadi panjang dan membingungkan

## Konsep Sub Program

### Tipe Sub Program

1. **Fungsi (Function)**
   - Mengembalikan nilai setelah eksekusi
   - Memiliki tipe return yang spesifik
   - Dapat digunakan dalam ekspresi

2. **Prosedur (Procedure)**
   - Tidak mengembalikan nilai (void)
   - Digunakan untuk melakukan aksi/operasi
   - Dipanggil sebagai statement terpisah

### Anatomi Sub Program
```cpp
tipe_return nama_fungsi(parameter1, parameter2, ...) {
    // Deklarasi variabel lokal
    // Algoritma/logic
    return nilai; // (untuk fungsi non-void)
}
```

---

## Fungsi (Non-Void)

Fungsi yang mengembalikan nilai dan dapat digunakan dalam ekspresi.

### Contoh Sederhana
```cpp
#include <iostream>
using namespace std;

// Fungsi yang mengembalikan nilai 1000
int mintaSeribu() {
    return 1000;
}

int main() {
    int i;
    cout << "Contoh pembuatan fungsi" << endl;
    i = mintaSeribu();  // Memanggil fungsi dan menyimpan hasilnya
    cout << "Nilai dari variabel i = " << i << endl;
    return 0;
}
```

### Fungsi dengan Parameter
```cpp
// Fungsi untuk perkalian dua bilangan
int perkalian(int bil_1, int bil_2) {
    int hasil;
    hasil = bil_1 * bil_2;
    return hasil;
}

int main() {
    int a = 5, b = 10, c;
    c = perkalian(a, b);  // Memanggil fungsi dengan parameter
    cout << "Hasil perkalian a * b = " << c << endl;
    return 0;
}
```

### Contoh Fungsi Praktis
```cpp
// Fungsi untuk mencari nilai maksimum dari dua bilangan
int Max2(int a1, int b1) {
    if (a1 >= b1) {
        return a1;
    } else {
        return b1;
    }
}

// Fungsi untuk menghitung rata-rata tiga bilangan
float AVG(int a, int b, int c) {
    return (float)(a + b + c) / 3;
}

// Fungsi dengan logika kondisional
int fungsiAneh(int A, int B) {
    if (A > B) {
        return A * B;
    } else {
        return A - B;
    }
}
```

## Prosedur (Void Function)

Prosedur adalah fungsi yang tidak mengembalikan nilai, digunakan untuk melakukan aksi atau operasi.

### Contoh Prosedur Sederhana
```cpp
#include <iostream>
using namespace std;

// Prosedur untuk mencetak garis
void cetakGaris() {
    cout << "-------------------------------------" << endl;
}

int main() {
    cetakGaris();
    cout << "Berikut ini adalah fungsi cetak garis" << endl;
    cetakGaris();
    return 0;
}
```

### Prosedur dengan Parameter
```cpp
// Prosedur untuk mencetak biodata
void tampilkanBiodata(string nama, int umur, string kota) {
    cout << "=== BIODATA ===" << endl;
    cout << "Nama: " << nama << endl;
    cout << "Umur: " << umur << " tahun" << endl;
    cout << "Kota: " << kota << endl;
    cout << "===============" << endl;
}

// Prosedur untuk mencetak tabel perkalian
void tabelPerkalian(int n) {
    cout << "Tabel Perkalian " << n << ":" << endl;
    for(int i = 1; i <= 10; i++) {
        cout << n << " x " << i << " = " << (n * i) << endl;
    }
}
```

## Parameter dalam Sub Program

### Jenis Parameter

1. **Parameter Formal**
   - Variabel yang dideklarasikan dalam definisi fungsi
   - Berfungsi sebagai placeholder untuk nilai yang akan diterima

2. **Parameter Aktual**
   - Nilai atau variabel yang dikirimkan saat memanggil fungsi
   - Nilai yang sebenarnya diproses oleh fungsi

```cpp
int tambah(int x, int y) {  // x dan y adalah parameter formal
    return x + y;
}

int main() {
    int a = 5, b = 3;
    int hasil = tambah(a, b);  // a dan b adalah parameter aktual
    return 0;
}
```

### Passing Parameter

#### 1. Pass by Value
```cpp
void ubahNilai(int x) {
    x = x + 10;  // Hanya mengubah salinan, tidak mempengaruhi variabel asli
    cout << "Dalam fungsi, x = " << x << endl;
}

int main() {
    int angka = 5;
    ubahNilai(angka);
    cout << "Setelah fungsi, angka = " << angka << endl;  // Tetap 5
    return 0;
}
```

#### 2. Pass by Reference
```cpp
void ubahNilaiRef(int &x) {
    x = x + 10;  // Mengubah variabel asli
    cout << "Dalam fungsi, x = " << x << endl;
}

int main() {
    int angka = 5;
    ubahNilaiRef(angka);
    cout << "Setelah fungsi, angka = " << angka << endl;  // Menjadi 15
    return 0;
}
```

## Contoh Implementasi Lengkap

### Program Kalkulator dengan Sub Program
```cpp
#include <iostream>
using namespace std;

// Fungsi-fungsi operasi matematika
float tambah(float a, float b) {
    return a + b;
}

float kurang(float a, float b) {
    return a - b;
}

float kali(float a, float b) {
    return a * b;
}

float bagi(float a, float b) {
    if (b != 0) {
        return a / b;
    } else {
        cout << "Error: Pembagian dengan nol!" << endl;
        return 0;
    }
}

// Prosedur untuk menampilkan menu
void tampilkanMenu() {
    cout << "\n=== KALKULATOR SEDERHANA ===" << endl;
    cout << "1. Penjumlahan" << endl;
    cout << "2. Pengurangan" << endl;
    cout << "3. Perkalian" << endl;
    cout << "4. Pembagian" << endl;
    cout << "5. Keluar" << endl;
    cout << "Pilih operasi: ";
}

int main() {
    int pilihan;
    float num1, num2, hasil;
    
    do {
        tampilkanMenu();
        cin >> pilihan;
        
        if (pilihan >= 1 && pilihan <= 4) {
            cout << "Masukkan bilangan pertama: ";
            cin >> num1;
            cout << "Masukkan bilangan kedua: ";
            cin >> num2;
            
            switch(pilihan) {
                case 1:
                    hasil = tambah(num1, num2);
                    cout << "Hasil: " << num1 << " + " << num2 << " = " << hasil << endl;
                    break;
                case 2:
                    hasil = kurang(num1, num2);
                    cout << "Hasil: " << num1 << " - " << num2 << " = " << hasil << endl;
                    break;
                case 3:
                    hasil = kali(num1, num2);
                    cout << "Hasil: " << num1 << " ร " << num2 << " = " << hasil << endl;
                    break;
                case 4:
                    hasil = bagi(num1, num2);
                    if (num2 != 0) {
                        cout << "Hasil: " << num1 << " รท " << num2 << " = " << hasil << endl;
                    }
                    break;
            }
        }
    } while (pilihan != 5);
    
    cout << "Terima kasih!" << endl;
    return 0;
}
```

## Latihan

### Latihan 1: Fungsi Rata-Rata (AVG)
**Objektif:** Tuliskan fungsi `AVG`

**Spesifikasi:**
- **Input**: Menerima **tiga bilangan bulat**.
- **Output**: Mengembalikan **nilai rata-rata** dari ketiga bilangan tersebut.

### Latihan 2: Fungsi Aneh

**Objektif:** Tuliskan fungsi `Aneh`

**Spesifikasi:**
- **Input**: Menerima **dua bilangan bulat**
- **Output**: Jika A > B, kembalikan hasil A * B. Jika B > A, kembalikan hasil A - B.

---

## Catatan

- **Penamaan Fungsi**: Gunakan nama yang deskriptif dan menggambarkan fungsi yang dilakukan
- **Parameter**: Berikan parameter yang cukup untuk fungsi bekerja optimal
- **Return Value**: Pastikan fungsi selalu mengembalikan nilai yang valid
