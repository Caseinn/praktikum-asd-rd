Array adalah struktur data fundamental yang memungkinkan penyimpanan banyak data dengan tipe yang sama dalam satu nama variabel.  
**Array** adalah kumpulan nilai yang memiliki tipe serupa yang disimpan dengan nama yang sama, dimana setiap elemen memiliki index sebagai pengenal.  
**Index** adalah nomor urut yang dimulai dari 0 untuk mengakses elemen tertentu dalam array.

Tanpa array, programmer harus membuat variabel terpisah untuk setiap data, yang membuat kode tidak efisien dan sulit dikelola.

---

## Tujuan Pembelajaran

Setelah mempelajari materi ini, mahasiswa diharapkan mampu:

- **Memahami konsep array** sebagai solusi penyimpanan data yang efisien dibandingkan menggunakan banyak variabel terpisah.
- **Mendeklarasikan dan menggunakan array** dengan sintaks yang benar sesuai aturan bahasa pemrograman C++.
- **Mengakses elemen array** menggunakan index, baik secara langsung maupun dengan perulangan.
- **Menerapkan array dalam program** untuk menyelesaikan masalah yang melibatkan kumpulan data dengan tipe yang sama.

---

## Mengapa Penting?

1. **Efisiensi Kode**  
   Dibandingkan membuat variabel `nama1, nama2, nama3, ..., nama100`, array memungkinkan penyimpanan dengan satu nama variabel seperti `nama[100]`. Ini membuat kode lebih ringkas dan mudah dipelihara.

2. **Kemudahan Pemrosesan**  
   Array dapat diproses dengan perulangan, memungkinkan operasi pada ratusan atau ribuan data dengan kode yang sederhana. Tanpa array, kita harus menulis kode berulang-ulang.

3. **Fondasi Struktur Data Lanjutan**  
   Array merupakan dasar untuk memahami struktur data yang lebih kompleks seperti matriks, stack, queue, dan hash table.

## Masalah Tanpa Array

Bayangkan membuat program untuk mengelola 10 nama siswa:

```cpp
// Tanpa array - tidak efisien
string nama1, nama2, nama3, nama4, nama5;
string nama6, nama7, nama8, nama9, nama10;

cin >> nama1 >> nama2 >> nama3; // ... hingga nama10
cout << nama1 << " - " << nama2 << endl;
cout << nama1 << " - " << nama3 << endl;
// ... dan seterusnya untuk semua kombinasi
```

**Masalah:**
- Kode menjadi sangat panjang dan repetitif
- Sulit melakukan operasi pada semua data sekaligus
- Tidak praktis untuk data dalam jumlah besar

## Konsep Array

### Definisi dan Karakteristik

Array memiliki karakteristik utama:
- **Homogen**: Semua elemen memiliki tipe data yang sama
- **Statis**: Ukuran array harus ditentukan saat deklarasi
- **Berindex**: Setiap elemen diakses melalui index (dimulai dari 0)
- **Random Access**: Elemen dapat diakses secara langsung atau berurutan

### Visualisasi Array

## Deklarasi Array

### Sintaks Dasar
```cpp
<tipe_data> <nama_array>[<jumlah_elemen>];
```

### Contoh Deklarasi
```cpp
int tinggi[10];      // Array 10 bilangan bulat (index 0-9)
float sesuatu[5];    // Array 5 bilangan desimal (index 0-4)
char huruf[100];     // Array 100 karakter (index 0-99)
```

---

## Pengisian dan Akses Array

### Pengisian Langsung
```cpp
tinggi[0] = 165;
tinggi[1] = 155;
tinggi[2] = 163;
```

### Pengisian dengan Perulangan
```cpp
for(int i = 0; i < 10; i++) {
    cin >> nilai[i];  // Input dari user
    // atau
    nilai[i] = i * 10;  // Nilai terkalkulasi
}
```

### Inisialisasi saat Deklarasi
```cpp
int tinggi[5] = {165, 155, 163, 170, 155};
```

### Mengakses Elemen
```cpp
// Akses langsung
cout << nilai[5];

// Akses dengan perulangan
for(int i = 0; i < 10; i++) {
    cout << nilai[i] << " ";
}
```

## Array Terisi Sebagian

Dalam praktik nyata, array tidak selalu terisi penuh. Kita menggunakan konsep **nilai efektif (Neff)**:

```cpp
int data[100];  // Kapasitas 100 elemen
int Neff = 6;   // Hanya 6 elemen yang terisi

// Hanya memproses elemen yang terisi
for(int i = 0; i < Neff; i++) {
    cout << data[i] << " ";
}
```

### Contoh Program Lengkap
```cpp
#include <iostream>
using namespace std;

int main() {
    int TabInt[100], Neff;
    
    cout << "Masukkan jumlah data: ";
    cin >> Neff;
    
    // Mengisi array
    for(int i = 0; i < Neff; i++) {
        cout << "Data ke-" << i+1 << ": ";
        cin >> TabInt[i];
    }
    
    // Menampilkan array
    cout << "Data yang dimasukkan: ";
    for(int i = 0; i < Neff; i++) {
        cout << TabInt[i] << " ";
    }
    
    return 0;
}
```

## Algoritma Pencarian (Searching)

Pencarian adalah operasi fundamental pada array:

```cpp
// Mencari nilai X dalam array
bool found = false;
int index = -1;

for(int i = 0; i < Neff && !found; i++) {
    if(TabInt[i] == X) {
        found = true;
        index = i;
    }
}

if(found) {
    cout << "Data ditemukan pada index: " << index;
} else {
    cout << "Data tidak ditemukan";
}
```

---

## Catatan

- **Index dimulai dari 0**: Array dengan 10 elemen memiliki index 0-9, bukan 1-10.
- **Jaga batas array**: Mengakses index di luar batas dapat menyebabkan error atau hasil tidak terduga.
- **Gunakan Neff**: Untuk array yang tidak terisi penuh, selalu gunakan nilai efektif dalam perulangan.
- **Penamaan yang jelas**: Gunakan nama array yang mendeskripsikan isinya, seperti `nilaiSiswa` atau `dataSuhu`.
