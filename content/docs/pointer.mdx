Pointer adalah variabel yang menyimpan alamat (referensi) dari nilai lain, berbeda dengan variabel biasa yang menyimpan nilai miliknya sendiri.  
**Pointer** adalah variabel yang berisi alamat memori dari variabel lain, memungkinkan akses tidak langsung ke data.  
**Reference** adalah proses mendapatkan alamat memori suatu variabel menggunakan operator `&`.  
**Dereference** adalah proses mengakses nilai yang ditunjuk oleh pointer menggunakan operator `*`.

Tanpa pointer, programmer kesulitan untuk membuat fungsi yang dapat memodifikasi nilai variabel asli, atau mengelola memori secara efisien.

---

## Tujuan Pembelajaran

Setelah mempelajari materi ini, mahasiswa diharapkan mampu:

- **Memahami konsep pointer** sebagai variabel yang menyimpan alamat memori dan perbedaannya dengan variabel biasa.
- **Mendeklarasikan dan mengoperasikan pointer** dengan sintaks yang benar menggunakan operator `*` dan `&`.
- **Mengimplementasikan passing by reference** dalam sub program untuk memodifikasi nilai variabel asli.
- **Menerapkan pointer dalam program** untuk menyelesaikan masalah yang memerlukan manipulasi alamat memori.

---

## Mengapa Penting?

1. **Modifikasi Nilai Asli**  
   Pointer memungkinkan fungsi untuk mengubah nilai variabel asli, tidak hanya salinannya. Ini sangat berguna untuk operasi seperti swap, sorting, dan manipulasi data kompleks.

2. **Efisiensi Memori**  
   Dengan pointer, kita dapat menghindari penyalinan data besar. Sebagai gantinya, kita hanya mengirim alamat memorinya, menghemat waktu dan ruang.

3. **Fondasi Konsep Lanjutan**  
   Pointer merupakan dasar untuk memahami konsep seperti linked list, dynamic memory allocation, dan struktur data yang lebih kompleks.

---

## Masalah Tanpa Pointer

Bayangkan membuat fungsi untuk menukar dua bilangan:

```cpp
// Tanpa pointer - tidak berhasil menukar nilai asli
void tukar(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    tukar(x, y);
    cout << x << " " << y;  // Output: 10 20 (tidak berubah!)
    return 0;
}
```

**Masalah:**
- Fungsi hanya bekerja pada salinan nilai (passing by value)
- Nilai asli variabel tidak berubah
- Tidak efisien untuk data berukuran besar

## Konsep Pointer

### Definisi dan Karakteristik

Pointer memiliki karakteristik utama:
- **Menyimpan Alamat**: Berisi alamat memori, bukan nilai langsung
- **Typed**: Memiliki tipe data yang sesuai dengan variabel yang ditunjuk
- **Dapat Dideference**: Menggunakan operator `*` untuk mengakses nilai yang ditunjuk
- **Dapat Direference**: Menggunakan operator `&` untuk mendapatkan alamat variabel

### Visualisasi Pointer
```
Memori:
┌─────────┬─────────┬─────────┬─────────┐
│ Alamat  │  1000   │  1004   │  1008   │
├─────────┼─────────┼─────────┼─────────┤
│ Variabel│    x    │   px    │  lain   │
├─────────┼─────────┼─────────┼─────────┤
│  Nilai  │   13    │  1000   │   ...   │
└─────────┴─────────┴─────────┴─────────┘
           ↑         ↑
           │         └── px menunjuk ke alamat 1000
           └── x berisi nilai 13
```

## Deklarasi Pointer

### Sintaks Dasar
```cpp
<tipe_data> *<nama_pointer>;
```

### Contoh Deklarasi dan Inisialisasi
```cpp
int x = 13;        // Variabel biasa
int *px;           // Deklarasi pointer ke int
px = &x;           // px menunjuk ke alamat x

// Atau sekaligus:
int *py = &x;      // Deklarasi dan inisialisasi
```

---

## Operasi Dasar Pointer

### Operator Reference (&)
```cpp
int x = 42;
cout << &x;        // Menampilkan alamat x (misal: 0x7fff5fbff61c)
```

### Operator Dereference (*)
```cpp
int x = 42;
int *px = &x;
cout << *px;       // Menampilkan nilai yang ditunjuk px (42)
*px = 100;         // Mengubah nilai x menjadi 100 melalui pointer
```

### Contoh Program Lengkap
```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 12;
    int *px;
    
    cout << "Alamat x : " << &x << endl;
    cout << "Nilai x : " << x << endl;
    
    px = &x;
    
    cout << "Variabel px berisi alamat x : " << px << endl;
    cout << "Nilai yang diacu px : " << *px << endl;
    
    *px = 25;  // Mengubah nilai x melalui pointer
    cout << "Nilai x setelah diubah melalui px : " << x << endl;
    
    return 0;
}
```

## Passing by Reference

### Konsep Passing Parameter

**Passing by Value**: Parameter adalah salinan nilai asli
```cpp
void ubah(int sesuatu) {
    sesuatu = sesuatu + 100;  // Hanya mengubah salinan
}
```

**Passing by Reference**: Parameter adalah alamat variabel asli
```cpp
void ubah(int *sesuatu) {
    *sesuatu = *sesuatu + 100;  // Mengubah nilai asli
}
```

### Perbandingan Implementasi

```cpp
// Passing by Value
void ubahByValue(int nilai) {
    nilai = nilai * 2;
}

// Passing by Reference
void ubahByReference(int *nilai) {
    *nilai = *nilai * 2;
}

int main() {
    int x = 10;
    
    ubahByValue(x);
    cout << x;  // Output: 10 (tidak berubah)
    
    ubahByReference(&x);
    cout << x;  // Output: 20 (berubah)
    
    return 0;
}
```

## Fungsi dengan Pointer

### Fungsi yang Mengembalikan Nilai
```cpp
int tambah(int *a, int b) {
    return *a + b;  // Menggunakan nilai yang ditunjuk a
}

int main() {
    int x = 15, y = 20;
    cout << tambah(&x, y);  // Output: 35
    return 0;
}
```

### Prosedur Tukar (Swap)
```cpp
void tukar(int *x, int *y) {
    int temp = *x;
    *x = *y;
    *y = temp;
}

int main() {
    int a = 10, b = 20;
    cout << "Sebelum: a=" << a << ", b=" << b << endl;
    
    tukar(&a, &b);
    cout << "Sesudah: a=" << a << ", b=" << b << endl;
    
    return 0;
}
```

## Algoritma dengan Pointer

### Pencarian dengan Pointer
```cpp
bool cari(int *arr, int n, int target, int *index) {
    for(int i = 0; i < n; i++) {
        if(arr[i] == target) {
            *index = i;
            return true;
        }
    }
    *index = -1;
    return false;
}
```

### Pengurutan dengan Pointer
```cpp
void insertionSort(int *arr, int n) {
    for(int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        
        while(j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}
```

## Pointer dengan Array

### Array sebagai Pointer
```cpp
int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr;  // arr sama dengan &arr[0]

// Dua cara akses yang equivalen:
cout << arr[2];   // Akses array biasa
cout << *(ptr+2); // Akses melalui pointer arithmetic
```

### Fungsi dengan Array Parameter
```cpp
void cetakArray(int *arr, int n) {
    for(int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
}

// Atau equivalent:
void cetakArray(int arr[], int n) {
    for(int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
}
```

---

## Catatan

- **Inisialisasi Pointer**: Selalu inisialisasi pointer sebelum digunakan. Pointer yang tidak diinisialisasi dapat menunjuk ke alamat sembarang.
- **Null Pointer**: Gunakan `nullptr` (C++11) atau `NULL` untuk pointer yang tidak menunjuk ke mana-mana.
